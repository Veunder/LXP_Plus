# Бэкенд: учёт ноутбуков

Flask + SQLAlchemy + PostgreSQL. REST API для фронтенда на Vue.

Таблицы: `teachers` (преподаватели), `laptops` (ноутбуки), `history` (журнал бронирований).

## Запуск

1. Создать базу в PostgreSQL:
   ```sql
   CREATE DATABASE laptops_db;
   ```
2. Окружение и зависимости:
   ```bash
   python -m venv venv
   source venv/bin/activate        # Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```
3. Настроить подключение:
   ```bash
   cp .env.example .env             # вписать свой логин/пароль/имя базы
   ```
4. Создать таблицы и тестовые данные:
   ```bash
   python seed.py
   ```
5. Запустить сервер:
   ```bash
   python app.py                    # http://localhost:5000
   ```

## Эндпоинты

| Метод  | Адрес                        | Что делает                                   |
|--------|------------------------------|----------------------------------------------|
| GET    | /api/stats                   | сводка: всего / свободны / заняты            |
| GET    | /api/teachers                | список преподавателей (для выпадающего списка)|
| GET    | /api/laptops                 | список (?status= , ?nclass= , ?q= поиск)     |
| POST   | /api/laptops                 | добавить ноутбук                             |
| GET    | /api/laptops/<id>            | один ноутбук + его история                   |
| PUT    | /api/laptops/<id>            | изменить                                     |
| DELETE | /api/laptops/<id>            | удалить                                       |
| POST   | /api/laptops/<id>/book       | взять (тело: id_pred / id_stud, stime, ftime)|
| POST   | /api/laptops/<id>/return     | вернуть                                       |
| GET    | /api/history                 | журнал броней (?active=true — только активные)|

## Пример: взять ноутбук

```json
POST /api/laptops/1/book
{
  "id_pred": 1,
  "nclass": "А-101",
  "stime": "10:30",
  "mouse": true,
  "charger": true
}
```
